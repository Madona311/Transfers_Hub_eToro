# Transfers Hub

This workspace runs the provided JSX dashboard as a standalone web page without a Node.js toolchain.

## Run locally

1. Open a terminal in this folder.
2. Start a static server, for example:
   - Python: `python -m http.server 5500`
3. Open http://localhost:5500 in your browser.

## Notes

- Source component: `app.jsx`
- Entry page: `index.html`
- React and Babel are loaded from CDN for local preview.
